# WEBSITE THÀNH PHỐ ĐÀ NẴNG 
## Cấu trúc thư mục đề xuất:
da-nang-city-website/
├── index.html
├── style.css
├── script.js
└── images/ (thư mục dành cho ảnh local nếu cần)
### 1. Cách triển khai:
1. Tạo thư mục mới cho dự án
2. Tạo 3 file: index.html, style.css, script.js
3. Copy nội dung từng file vào đúng file tương ứng
4. Mở file index.html bằng trình duyệt để xem kết quả
### 2. Chức năng JavaScript đã tích hợp:
- Điều hướng mượt khi click vào menu
- Highlight menu item khi scroll
- Form liên hệ với validation cơ bản
- Hiệu ứng load ảnh động
- Nút cuộn lên đầu trang
- Hiệu ứng fade-in khi scroll
- Responsive mobile menu
- Thông báo tương tác cho nút bấm
### 3. Tùy chỉnh nếu cần:
#### Để thay thế ảnh bằng ảnh local:
1. Tải ảnh về và đặt trong thư mục `images/`
2. Trong style.css, thay đổi các URL ảnh từ:
   `background-image: url('https://picsum.photos/...')`
   thành:
   `background-image: url('images/ten-anh-cua-ban.jpg')`
### 4. Tính năng có thể mở rộng:
1. Thêm nhiều địa điểm du lịch
2. Thêm bản đồ Google Maps
3. Thêm gallery ảnh
4. Thêm chế độ dark mode
5. Thêm ngôn ngữ tiếng Anh
